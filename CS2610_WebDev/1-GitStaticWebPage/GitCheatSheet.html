Philip's git Cheat Sheet

This is a reference table of git commands

git references
Command Description Flags Example
clone Clone a repository into a new directory   
git clone "URL"
add Stage a file contents for commit -n Dry Run: doesn't actually add the files
-u Update: only files currently tracked
-A All: all files in the project 
git add file.html
commit Record staged changes to the repository -m <msg> Message: use the given <msg> as the commit message 
git commit -m"commit message"
push Update remote repository with committed files origin <branch name> 
git push origin master

To learn more, checkout the git documentation
