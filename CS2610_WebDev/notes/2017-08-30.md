### HTML Tags
HyperText Markup Language

* \<tag> Content \</tag>
  * the whole expression is an element

#### Tags
* `<img>` : for inserting images
  * attributes : `src` and `alt`
  * `<img src="web address" alt="description"`
* `<h1>` : Headings 1-6
* `<p>` : Paragraph
* Lists
  * `<ol>` : Ordered / numbered list
  * `<ul>` : Unordered / bulleted lsit 
* `<strong>` : bold
* `<em` : emphasis / italics
